INTRODUCTION TO WEB DEV

Web development is the process of creating, building, and maintaining websites or web applications
that run on the internet (or intranet). It involves a mix of design, coding, and server management to make a website functional, user-friendly, and accessible.

LIST OF FRONT END LANGUAGE

  • HTML (HyperText Markup Language)
  • CSS (Cascading Style Sheets)
  • JavaScript (JS)

LIST OF FRONT END FRAME WORK AND LIBRARIES

  1. React.js
  2. Angular
  3. Vue.js
  4. Svelte
  5. jQuery
  6. Bootstrap
  7. Tailwind CSS

LIST OF BACK END LANGUAGE

JavaScript (Node.js)
Runs JavaScript on the server.
Python
Popular for simplicity and frameworks like Django & Flask.
PHP
Widely used, especially for WordPress and Laravel.
Java
Strong, scalable, used in Spring framework and enterprise apps.
C#
Used with .NET for web applications.
Ruby
Known for the Ruby on Rails framework.
Go (Golang)
Fast and efficient, great for high-performance systems.
C++
Less common for web, but used in performance-heavy back-end systems.
Kotlin
Often used for Android, but also works in back-end (e.g., with Ktor).
Scala
Runs on JVM, often used with the Play framework.
